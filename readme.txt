Dice Roll Game 🎲

This repository contains a simple dice roll game implemented using HTML, CSS, and JavaScript. The game allows two players to compete against each other, and the winner is determined by the roll of a virtual dice. 🏆

How to Play 🎮

Clone or download the repository to your local machine. 📥
Open the index.html file in your web browser. 🌐
The game starts with Player 1. To roll the dice, click on the "Click Me" button. 🎲
The dice will generate a random number between 1 and 6.
The player with the higher dice roll wins the game. 🥇
To play again, click on the "Play Again" button. 🔄


Tech Stack 🛠️
The following technologies were used to build this dice roll game:
HTML: Used to structure the web page.
CSS: Used to style the elements and layout of the game.
JavaScript: Used to handle the game logic and generate random dice numbers.
